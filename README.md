## Client Query Management System
A modern, full-stack ticketing and support dashboard built using Streamlit, PostgreSQL, Python, Pandas, and Plotly — designed to streamline client issue tracking and support management.

## Project Overview

The Client Query Management System provides an intuitive and centralized platform for clients to raise support queries and for support teams to manage, resolve, and analyze them efficiently.
With role-based login, smooth UI, dynamic visual dashboards, and CSV analytics for large datasets (5000+ records), CQMS is built for clarity, performance, and modern user experience.

## Run the application
streamlit run app.py
